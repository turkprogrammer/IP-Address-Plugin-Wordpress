== Description ==

The User IP Display plugin allows you to display the IP address of registered users in the WordPress admin area. It also saves the IP address of each user upon login and stores it in the database.

== Installation ==

1. Upload the 'user-ip-display' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= How do I view the IP address of users? =

After activating the plugin, the IP address column will be added to the Users table in the WordPress admin area. You can view the IP address of each user on the Users page.
